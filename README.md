# fin-rag — Фінансовий RAG-чатбот

Чатбот-радник на основі технології RAG (Retrieval-Augmented Generation), який відповідає на запитання про фінансові показники та HR-дані компаній. Використовує LangChain, OpenAI GPT-4o-mini та Chroma як векторну базу даних.

## Як це працює

1. Текстові документи з теки `resources/` завантажуються та розбиваються на фрагменти.
2. Фрагменти індексуються у Chroma через OpenAI Embeddings.
3. При кожному запиті система отримує найрелевантніші фрагменти та передає їх у GPT-4o-mini разом із питанням.
4. Модель відповідає виключно на фінансові та HR-запити; запити поза цією темою відхиляються.

```
resources/*.txt  →  DocumentLoader  →  TextSplitter  →  ChromaDB
                                                              ↓
              запит користувача  →  Retriever  →  LLM (GPT-4o-mini)  →  відповідь
```

## Вимоги

- Python ≥ 3.12
- [uv](https://docs.astral.sh/uv/) — менеджер пакетів
- OpenAI API ключ

## Встановлення та запуск

### 1. Клонувати репозиторій

```bash
git clone <repo-url>
cd fin_rag
```

### 2. Встановити залежності через uv

```bash
uv sync
```

### 3. Запустити Jupyter Notebook

```bash
uv run jupyter notebook app.ipynb
```

або у JupyterLab:

```bash
uv run jupyter lab app.ipynb
```

### 4. Ввести OpenAI API ключ

У першій клітинці ноутбука введіть свій ключ, коли з'явиться запит.

## Структура проєкту

```
fin_rag/
├── app.ipynb              # Основний ноутбук із RAG-пайплайном
├── pyproject.toml         # Конфігурація проєкту та залежності
├── uv.lock                # Зафіксовані версії залежностей
├── resources/             # Фінансові дані у форматі .txt
│   └── fin_monitoring.txt # Дані 4 українських компаній за 2024 рік
└── tools/                 # Модулі-утиліти
    ├── llm.py             # Ініціалізація LLM
    └── util.py            # Завантаження документів, векторна БД, форматування
```

## Дані

За замовчуванням `resources/fin_monitoring.txt` містить фінансові показники чотирьох українських компаній за 2024 рік:

| Компанія | Сектор | Виручка | ROE |
|---|---|---|---|
| ТехНова Інк | IT / SaaS | 450 млн грн | 15.2% |
| ГрінЕнерджі Солюшнс | Зелена енергетика | 890 млн грн | 19.7% |
| РітейлМакс Груп | Роздрібна торгівля | 1.2 млрд грн | 9.3% |
| ФінТех Дайнамікс | FinTech | 230 млн грн | від'ємний |

Щоб додати свої дані, помістіть `.txt`-файли до теки `resources/` — вони підхопляться автоматично.
